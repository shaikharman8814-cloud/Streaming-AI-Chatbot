# Streaming-AI-Chatbot
Terminal AI chatbot with memory and streaming output.
 Streaming AI Chatbot

A simple terminal-based AI chatbot built with Python.

Features:

* Real-time streaming responses
* Conversation memory
* System prompts
* Interactive terminal chat

How it works:

1. User enters a message.
2. Message is stored in chat history.
3. AI generates a streamed response.
4. Response is displayed character by character.
5. Conversation history is preserved for future messages.

Tech Used:

* Python
* OpenAI-compatible API
* Streaming Responses

Run:

python chatbot.py
